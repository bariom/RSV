# Demo Web

Questa cartella contiene una prima demo statica del sito dedicato alla storia di Riva San Vitale.

## File

- `index.html`: struttura della pagina;
- `styles.css`: stile della demo;
- `app.js`: contenuti iniziali renderizzati nel frontend.

## Avvio rapido

Puoi aprire direttamente `web/index.html` nel browser oppure servirlo con un web server locale.

## Obiettivo

La demo non e ancora collegata al database. Serve per:

- vedere una prima direzione visiva;
- capire come rendere timeline, luoghi, persone e fonti;
- preparare il passaggio successivo verso un backend o API che leggano i dati reali dal DB.
